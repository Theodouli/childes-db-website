Releases

childes-db will be updated every 6 months with a most recent parse of the CHILDES database, reflecting the addition of new corpora and the revision of older annotations (more details can be found in the documentation). Visualizations and the childesr package will always use the most recent database version by default. The R API, childesr, can be directed to use any recent database version by using the --version flag. We provide here .sql files for users wanting to refer to older releases or to install the database locally.
